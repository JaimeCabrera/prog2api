Api de Programacion 2

Rutas de tipo publica no require ningun tipo de header de autenticacion

Rutas privadas requieren del header del tipo x-access-token:"jwt-value"

Rutas del Usuario
Nombre Ruta Metodo Tipo
Crear /api/auth/signin POST Pública
Login /api/auth/signup POST Privada
Rutas de las categorías
Nombre Ruta Método Tipo
Crear categoría por usuario /api/categories POST Privada
Obtener las categorías por usuario /api/categories GET Privada
Obtener información de la categoria /api/categories/:categoryId {params:'categoryId'} GET Privada
Editar la categoría /api/categories/category {body:'categoryId'} PUT Privada
Eliminar la categoría /api/categories/:categoryId {params:'categoryId'} DELETE Privada
Rutas de las Tareas
Nombre Ruta Método Tipo
Crear tareas de una categoría /api/tasks {body:'categoryId'} POST Privada
Obtener todas las tareas por categoria /api/tasks/:categoryId {params:'categoryId'} GET Privada
Obtener información de una tarea por id /api/tasks {body:'taskId'} GET Privada
Editar la tarea /api/tasks/task {body:'taskId'} PUT Privada
Eliminar la tarea /api/tasks/:taskId {params:'taskId'} DELETE Privada
Cambiar el estado de una tarea /api/tasks/:taskId {params:'taskId'} PUT Privada
Rutas de las Tareas
Nombre Ruta Método Tipo
Enviar tareas por email /api/email {body:{'tasks','username','email','Destinatario'}} POST Privada
